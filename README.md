# FastAPI

新鋭のAPIサーバ用Webフレームワーク。python。

## Pythonの仮想環境

FastAPIに限った話ではないが。

とりあえず作る。

```
python -m venv .venv
```

有効化。bashの場合

```
source ./venv/scripts/activate
```

有効化。PowerShellの場合

```
./.venv/Scripts/Activate.ps1
```
